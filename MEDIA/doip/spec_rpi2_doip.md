# 라즈베리파이#2 (0x0030) — 미디어 장치 진단 (수정)

## UDS 서비스:

| SID  | 서비스                     | 지원 범위                     |
|------|----------------------------|-------------------------------|
| 0x10 | DiagnosticSessionControl   | 0x01 (Default)                |
| 0x22 | ReadDataByIdentifier       | 0xF190, 0x0300~0x0306         |
| 0x19 | ReadDTCInformation         | 0x02 (ByStatusMask)           |
| 0x14 | ClearDiagnosticInformation | 0xFFFFFF (전체 삭제)           |

---

## DID:

| DID    | 읽는 값              | 바이트 | 값 범위          | 확인 방법                                                                     |
|--------|----------------------|--------|------------------|-------------------------------------------------------------------------------|
| 0xF190 | 노드 식별 이름       | 4B     | "MED1"           | 고정값                                                                        |
| 0x0300 | eth0 상태            | 1B     | 0=DOWN, 1=UP     | `ip link show eth0` → state UP/DOWN                                           |
| 0x0301 | 전방 카메라 상태     | 1B     | 0=미연결, 1=연결 | `/dev/video0` USB 카메라 존재 및 스트림 확인                                  |
| 0x0302 | 후방 카메라 상태     | 1B     | 0=미연결, 1=연결 | `/dev/video2` 존재 여부                                                       |
| 0x0303 | 마이크 상태          | 1B     | 0=미연결, 1=연결 | `/proc/asound/card0` 존재 여부                                                |
| 0x0304 | Flask 서버 상태      | 1B     | 0=중지, 1=실행 중| TCP 8080 포트 listen 여부                                                     |
| 0x0305 | SOME/IP 서버 상태    | 1B     | 0=중지, 1=실행 중| UDP 30491 포트 listen 여부                                                    |
| 0x0306 | 저장공간 사용률 (%)  | 1B     | 0~100            | `df` 명령으로 확인                                                            |

---

## DTC:

| DTC      | 의미              | 감지 조건                                                   | 복구 조건               |
|----------|-------------------|-------------------------------------------------------------|-------------------------|
| 0xC30000 | 전방 카메라 고장  | `/dev/video0` 사라짐                                        | 재연결 감지             |
| 0xC30100 | 후방 카메라 고장  | `/dev/video2` 사라짐 또는 스트리밍 불가                     | 재연결 및 스트리밍 복귀 |
| 0xC30200 | 마이크 고장       | `/proc/asound/card0` 또는 `pcm0c` 사라짐                    | 재연결 감지             |
| 0xC30300 | 저장공간 부족     | 사용률 > 90%                                                | 사용률 ≤ 90%            |
| 0xC30400 | Flask 서버 오류   | TCP 8080 listen 안 됨                                       | listen 재개             |
| 0xC30500 | SOME/IP 서버 오류 | UDP 30491 listen 안 됨                                      | listen 재개             |
| 0xC30600 | eth0 통신 오류    | eth0 인터페이스 DOWN                                        | 인터페이스 UP           |
